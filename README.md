# 🖥️ Mein Raum — Outil de gestion réseau SIO/SISR

![Version](https://img.shields.io/badge/version-2.4.1-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey.svg)
![License](https://img.shields.io/badge/license-Proprietary-red.svg)

**Mein Raum** est une plateforme professionnelle de supervision, de gestion de VLAN et de reporting réseau, conçue spécifiquement pour les infrastructures administrées par les filières BTS SIO (Services Informatiques aux Organisations) et SISR (Solutions d'Infrastructure, Systèmes et Réseaux).

> ⚠️ **ATTENTION :** Ce dépôt est réservé à la distribution des clients lourds pour les serveurs de production. L'accès aux serveurs d'administration nécessite une authentification stricte.

---

## 📥 Téléchargements (Version 2.4.1)

Les installeurs sont disponibles dans la section [**Releases**](../../releases/latest) de ce dépôt. 

* 🪟 **Windows (10/11) :** `meinraum-setup.exe`
* 🐧 **Linux (Debian/Ubuntu) :** `meinraum_2.4.1_amd64.deb`
* 🍏 **macOS (13+) :** `meinraum-2.4.1.dmg`
